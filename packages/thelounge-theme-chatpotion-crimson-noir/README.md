# ChatPotion Crimson Noir

Crimson Noir theme from **ChatPotion** for [The Lounge](https://thelounge.chat/) IRC client.

## Installation

Install this theme with The Lounge:

```sh
thelounge install thelounge-theme-chatpotion-crimson-noir
```

Then open The Lounge settings and select:

```text
ChatPotion Crimson Noir
```

## Recommended Base Theme

This theme works best when used with The Lounge's **Morning** base theme.
